# Página n9 - Análise de Variância (ANOVA)

## ANOVA de Um Fator

### Modelo
$Y_{ij} = \mu + \alpha_i + \varepsilon_{ij}$

onde:
- $\mu$ = média geral
- $\alpha_i$ = efeito do tratamento $i$
- $\varepsilon_{ij} \sim N(0, \sigma^2)$

### Hipóteses
- $H_0: \alpha_1 = \alpha_2 = \cdots = \alpha_k = 0$
- $H_1$: Pelo menos um $\alpha_i \neq 0$

### Tabela ANOVA
| Fonte | SQ | gl | QM | F |
|-------|----|----|----|---|
| Entre grupos | SQE | $k-1$ | QME | QME/QMR |
| Dentro grupos | SQR | $N-k$ | QMR | |
| Total | SQT | $N-1$ | | |

### Estatística de Teste
$$F = \frac{\text{QME}}{\text{QMR}} \sim F_{k-1, N-k}$$

## Somas de Quadrados

### SQ Total
$$SQT = \sum_{i=1}^k \sum_{j=1}^{n_i} (Y_{ij} - \bar{Y})^2$$

### SQ Entre Grupos
$$SQE = \sum_{i=1}^k n_i (\bar{Y}_i - \bar{Y})^2$$

### SQ Dentro Grupos
$$SQR = \sum_{i=1}^k \sum_{j=1}^{n_i} (Y_{ij} - \bar{Y}_i)^2$$

## Pressupostos
1. Normalidade dos resíduos
2. Homocedasticidade
3. Independência das observações
