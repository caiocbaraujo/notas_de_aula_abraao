# Página n15 - Análise de Sobrevivência

## Conceitos Fundamentais

### Função de Sobrevivência
$$S(t) = P(T > t) = 1 - F(t)$$

onde $T$ = tempo até o evento.

### Função de Risco (Hazard)
$$h(t) = \lim_{\Delta t \to 0} \frac{P(t \leq T < t + \Delta t | T \geq t)}{\Delta t}$$

### Relação entre S(t) e h(t)
$$S(t) = \exp\left(-\int_0^t h(u) du\right)$$

## Estimador de Kaplan-Meier

### Fórmula
$$\hat{S}(t) = \prod_{t_i \leq t} \left(1 - \frac{d_i}{n_i}\right)$$

onde:
- $d_i$ = número de eventos no tempo $t_i$
- $n_i$ = número de indivíduos em risco no tempo $t_i$

### Variância Estimada
$$\text{Var}[\hat{S}(t)] = [\hat{S}(t)]^2 \sum_{t_i \leq t} \frac{d_i}{n_i(n_i - d_i)}$$

## Teste Log-Rank

### Hipóteses
- $H_0$: Funções de sobrevivência são iguais
- $H_1$: Funções de sobrevivência diferem

### Estatística
$$\chi^2 = \frac{(O_1 - E_1)^2}{V_1} + \frac{(O_2 - E_2)^2}{V_2}$$

onde $O_i$ = eventos observados, $E_i$ = eventos esperados, $V_i$ = variância.

## Modelo de Cox

### Hazard Proporcional
$$h(t|x) = h_0(t) \exp(\beta^T x)$$

onde $h_0(t)$ = hazard baseline.
