# Página n5 - Estimação Pontual

## Conceitos Fundamentais

### Estimador
Uma estatística $T(X_1, \ldots, X_n)$ usada para estimar um parâmetro $\theta$.

### Estimativa
O valor numérico $t = T(x_1, \ldots, x_n)$ obtido de uma amostra específica.

## Propriedades dos Estimadores

### Não-Viciado (Não-Tendencioso)
$E[T] = \theta$ para todo $\theta$.

### Consistente
$T_n \xrightarrow{P} \theta$ quando $n \to \infty$.

### Eficiente
Entre estimadores não-viciados, aquele com menor variância.

## Métodos de Estimação

### Método dos Momentos
Igualar momentos amostrais aos momentos populacionais:

$$\frac{1}{n}\sum_{i=1}^n X_i^k = E[X^k]$$

### Máxima Verossimilhança
Encontrar $\hat{\theta}$ que maximiza:

$$L(\theta) = \prod_{i=1}^n f(x_i; \theta)$$

ou equivalentemente:

$$\ell(\theta) = \sum_{i=1}^n \log f(x_i; \theta)$$

## Exemplos Clássicos

### Média Amostral
$\bar{X} = \frac{1}{n}\sum_{i=1}^n X_i$ é estimador não-viciado e consistente de $\mu$.

### Variância Amostral
$S^2 = \frac{1}{n-1}\sum_{i=1}^n (X_i - \bar{X})^2$ é não-viciada para $\sigma^2$.
