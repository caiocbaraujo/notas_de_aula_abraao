# Página n11 - Regressão Linear Múltipla

## Modelo Geral

### Equação do Modelo
$Y_i = \beta_0 + \beta_1 X_{1i} + \beta_2 X_{2i} + \cdots + \beta_p X_{pi} + \varepsilon_i$

### Notação Matricial
$\mathbf{Y} = \mathbf{X}\boldsymbol{\beta} + \boldsymbol{\varepsilon}$

onde:
- $\mathbf{Y} = (Y_1, \ldots, Y_n)^T$
- $\mathbf{X}$ = matriz de design $(n \times (p+1))$
- $\boldsymbol{\beta} = (\beta_0, \beta_1, \ldots, \beta_p)^T$
- $\boldsymbol{\varepsilon} \sim N(\mathbf{0}, \sigma^2 \mathbf{I})$

## Estimador dos Mínimos Quadrados

### Solução
$$\hat{\boldsymbol{\beta}} = (\mathbf{X}^T\mathbf{X})^{-1}\mathbf{X}^T\mathbf{Y}$$

### Propriedades
- $E[\hat{\boldsymbol{\beta}}] = \boldsymbol{\beta}$ (não-viciado)
- $\text{Var}(\hat{\boldsymbol{\beta}}) = \sigma^2(\mathbf{X}^T\mathbf{X})^{-1}$

## Testes de Hipóteses

### Teste Global
- $H_0: \beta_1 = \beta_2 = \cdots = \beta_p = 0$
- $H_1$: Pelo menos um $\beta_j \neq 0$

**Estatística**: $F = \frac{MSR}{MSE} \sim F_{p, n-p-1}$

### Testes Individuais
- $H_0: \beta_j = 0$
- $H_1: \beta_j \neq 0$

**Estatística**: $t = \frac{\hat{\beta}_j}{SE(\hat{\beta}_j)} \sim t_{n-p-1}$

## Multicolinearidade
Problema quando variáveis independentes são altamente correlacionadas, causando:
- Estimadores instáveis
- Erros padrão inflacionados
- Dificuldade na interpretação
