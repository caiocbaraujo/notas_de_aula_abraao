# Página n7 - Testes de Hipóteses

## Conceitos Básicos

### Hipóteses
- **Hipótese nula**: $H_0: \theta = \theta_0$
- **Hipótese alternativa**: $H_1: \theta \neq \theta_0$ (bilateral) ou $H_1: \theta > \theta_0$ (unilateral)

### Tipos de Erro
- **Erro Tipo I**: Rejeitar $H_0$ quando é verdadeira ($\alpha = P(\text{Erro I})$)
- **Erro Tipo II**: Aceitar $H_0$ quando é falsa ($\beta = P(\text{Erro II})$)

### Poder do Teste
$\pi(\theta) = 1 - \beta(\theta) = P(\text{Rejeitar } H_0 | \theta)$

## Testes para Média

### Teste Z (Variância Conhecida)
**Estatística**: $Z = \frac{\bar{X} - \mu_0}{\sigma/\sqrt{n}}$

**Região crítica**: $|Z| > z_{\alpha/2}$ (bilateral)

### Teste t (Variância Desconhecida)
**Estatística**: $t = \frac{\bar{X} - \mu_0}{S/\sqrt{n}}$

**Região crítica**: $|t| > t_{\alpha/2, n-1}$ (bilateral)

## Testes para Proporção

### Teste Binomial
**Estatística**: $Z = \frac{\hat{p} - p_0}{\sqrt{p_0(1-p_0)/n}}$

**Condições**: $np_0 \geq 5$ e $n(1-p_0) \geq 5$

## Valor-p
Probabilidade de observar uma estatística de teste pelo menos tão extrema quanto a observada, assumindo $H_0$ verdadeira.
