# Página n13 - Testes Não-Paramétricos

## Teste de Wilcoxon (Mann-Whitney)

### Aplicação
Comparar duas amostras independentes quando não se pode assumir normalidade.

### Hipóteses
- $H_0$: As duas populações têm a mesma distribuição
- $H_1$: As distribuições diferem em localização

### Procedimento
1. Combinar as duas amostras
2. Ordenar todas as observações
3. Calcular soma dos postos para cada grupo
4. Usar estatística $W$ ou $U$

### Estatística de Teste
$$U = n_1 n_2 + \frac{n_1(n_1+1)}{2} - R_1$$

onde $R_1$ = soma dos postos do grupo 1.

## Teste de Kruskal-Wallis

### Aplicação
ANOVA não-paramétrica para $k \geq 2$ grupos independentes.

### Hipóteses
- $H_0$: Todas as populações têm a mesma distribuição
- $H_1$: Pelo menos duas populações diferem

### Estatística
$$H = \frac{12}{N(N+1)} \sum_{i=1}^k \frac{R_i^2}{n_i} - 3(N+1)$$

onde $R_i$ = soma dos postos do grupo $i$.

### Distribuição
$H \sim \chi^2_{k-1}$ para amostras grandes.

## Teste de Correlação de Spearman

### Aplicação
Medir associação monotônica entre duas variáveis.

### Coeficiente
$$r_s = 1 - \frac{6\sum d_i^2}{n(n^2-1)}$$

onde $d_i$ = diferença entre os postos.
