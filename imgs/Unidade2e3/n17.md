# Página n17 - Métodos de Bootstrap

## Conceito Fundamental

### Bootstrap Não-Paramétrico
1. Extrair amostra bootstrap de tamanho $n$ com reposição da amostra original
2. Calcular estatística de interesse na amostra bootstrap
3. Repetir $B$ vezes
4. Usar distribuição empírica das $B$ estatísticas para inferência

### Estimador Bootstrap da Variância
$$\text{Var}^*(\hat{\theta}^*) = \frac{1}{B-1} \sum_{b=1}^B (\hat{\theta}_b^* - \bar{\theta}^*)^2$$

onde $\bar{\theta}^* = \frac{1}{B}\sum_{b=1}^B \hat{\theta}_b^*$.

## Intervalos de Confiança Bootstrap

### Método Percentil
$$CI_{1-\alpha} = [\hat{\theta}_{(\alpha/2)}, \hat{\theta}_{(1-\alpha/2)}]$$

onde $\hat{\theta}_{(p)}$ é o $p$-ésimo percentil da distribuição bootstrap.

### Método BCa (Bias-Corrected and Accelerated)
Ajusta para viés e assimetria da distribuição.

### Método t Bootstrap
$$CI = [\hat{\theta} - t_{1-\alpha/2} \cdot SE^*, \hat{\theta} + t_{1-\alpha/2} \cdot SE^*]$$

## Bootstrap Paramétrico

### Procedimento
1. Estimar parâmetros do modelo da amostra original
2. Gerar amostras bootstrap do modelo estimado
3. Aplicar método bootstrap padrão

### Vantagens
- Mais eficiente quando modelo é conhecido
- Melhor para pequenas amostras

## Aplicações
- Estimação de variância de estimadores complexos
- Intervalos de confiança para estatísticas não-padrão
- Testes de hipóteses
- Validação cruzada
