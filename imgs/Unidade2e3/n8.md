# Página n8 - Testes de Aderência

## Teste Qui-Quadrado de Aderência

### Hipóteses
- $H_0$: Os dados seguem a distribuição especificada
- $H_1$: Os dados não seguem a distribuição especificada

### Estatística de Teste
$$\chi^2 = \sum_{i=1}^k \frac{(O_i - E_i)^2}{E_i}$$

onde:
- $O_i$ = frequência observada na categoria $i$
- $E_i$ = frequência esperada na categoria $i$
- $k$ = número de categorias

### Região Crítica
Rejeitar $H_0$ se $\chi^2 > \chi^2_{\alpha, k-1-m}$

onde $m$ = número de parâmetros estimados.

## Teste de Independência

### Tabela de Contingência
Testar independência entre duas variáveis categóricas.

### Estatística
$$\chi^2 = \sum_{i=1}^r \sum_{j=1}^c \frac{(O_{ij} - E_{ij})^2}{E_{ij}}$$

onde $E_{ij} = \frac{R_i \cdot C_j}{n}$.

### Graus de Liberdade
$(r-1)(c-1)$ onde $r$ = número de linhas, $c$ = número de colunas.

## Condições para Aplicação
- Todas as frequências esperadas $\geq 5$
- Para amostras pequenas, usar teste exato de Fisher
