# Página n14 - Análise de Séries Temporais

## Componentes de uma Série Temporal

### Decomposição Clássica
$Y_t = T_t + S_t + C_t + \varepsilon_t$

onde:
- $T_t$ = tendência
- $S_t$ = sazonalidade
- $C_t$ = ciclo
- $\varepsilon_t$ = componente aleatória

## Modelos ARIMA

### Notação ARIMA(p,d,q)
- **AR(p)**: Autoregressivo de ordem $p$
- **I(d)**: Integrado de ordem $d$
- **MA(q)**: Média móvel de ordem $q$

### Modelo AR(1)
$$X_t = \phi X_{t-1} + \varepsilon_t$$

onde $|\phi| < 1$ para estacionariedade.

### Modelo MA(1)
$$X_t = \varepsilon_t + \theta \varepsilon_{t-1}$$

### Condições de Estacionariedade
- **AR(1)**: $|\phi| < 1$
- **AR(2)**: $\phi_1 + \phi_2 < 1$, $\phi_2 - \phi_1 < 1$, $|\phi_2| < 1$

## Função de Autocorrelação

### ACF
$$\rho_k = \frac{\gamma_k}{\gamma_0}$$

onde $\gamma_k = \text{Cov}(X_t, X_{t-k})$.

### PACF
Correlação parcial entre $X_t$ e $X_{t-k}$ removendo o efeito de $X_{t-1}, \ldots, X_{t-k+1}$.

## Identificação do Modelo
- **AR(p)**: PACF corta após lag $p$, ACF decai exponencialmente
- **MA(q)**: ACF corta após lag $q$, PACF decai exponencialmente
- **ARMA(p,q)**: Ambos decaem exponencialmente
