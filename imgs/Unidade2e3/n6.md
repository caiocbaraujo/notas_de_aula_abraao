# Página n6 - Intervalos de Confiança

## Definição

Um intervalo de confiança $(L, U)$ para $\theta$ com nível de confiança $(1-\alpha) \times 100\%$ satisfaz:

$$P(L \leq \theta \leq U) = 1 - \alpha$$

## Intervalos para Média Populacional

### Variância Conhecida
Se $X_1, \ldots, X_n \sim N(\mu, \sigma^2)$ com $\sigma^2$ conhecida:

$$\bar{X} \pm z_{\alpha/2} \frac{\sigma}{\sqrt{n}}$$

### Variância Desconhecida
Se $\sigma^2$ é desconhecida:

$$\bar{X} \pm t_{\alpha/2, n-1} \frac{S}{\sqrt{n}}$$

onde $S^2 = \frac{1}{n-1}\sum_{i=1}^n (X_i - \bar{X})^2$.

## Intervalos para Proporção

### Proporção Binomial
Se $X \sim \text{Binomial}(n,p)$:

$$\hat{p} \pm z_{\alpha/2} \sqrt{\frac{\hat{p}(1-\hat{p})}{n}}$$

onde $\hat{p} = \frac{X}{n}$.

### Correção para Amostras Pequenas
Para $n$ pequeno, usar distribuição binomial exata.

## Intervalos para Variância

### Distribuição Normal
Se $X_1, \ldots, X_n \sim N(\mu, \sigma^2)$:

$$\frac{(n-1)S^2}{\chi^2_{\alpha/2, n-1}} \leq \sigma^2 \leq \frac{(n-1)S^2}{\chi^2_{1-\alpha/2, n-1}}$$
