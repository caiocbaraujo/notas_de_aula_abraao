Há: $H_0: f_1(x) = f_0(x)$ e $H_1: f_1(x) = f_1(x)$.

\textbf{Solução:} Como o teste é do tipo simples x simples, podemos usar o LNP. Assim, temos que:

Rejeitamos $H_0$ se, e só se,
\begin{equation}
\frac{f_1(x)}{f_0(x)} > k.
\end{equation}

Note que
\begin{equation}
\frac{f_1(x)}{f_0(x)} = 4x^{-3/2}.
\end{equation}

O teste pode ser escrito como: rejeitamos $H_0$ para $x$ pertencente a
\begin{equation}
R_c = \left\{ x \in (0,4); \ 4x^{-3/2} > k \right\} \Rightarrow
\end{equation}
\begin{equation}
R_c = \left\{ x \in (0,4); \ x < \left( \frac{4}{k} \right)^{2/3} \right\} = \left\{ x \in (0,4); \ x < k_1 \right\}.
\end{equation}

Onde,
\begin{equation}
R_c = \left\{ x \in (0,4); \ x < k_1 \right\}.
\end{equation}
