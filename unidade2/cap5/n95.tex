

\textbf{Exercício:} Sejam $X_1, \ldots, X_n$ uma a.a de $X \sim N(\mu, \sigma^2)$ com $\mu \in \mathbb{R}$ e $\sigma^2 > 0$ desconhecido. Mostre que

\begin{equation}
IC_{\frac{1}{\alpha}}(\sigma^2) = \left[ \frac{(n-1)S_n^2}{\chi^2_{n-1;\alpha/2}}, \frac{(n-1)S_n^2}{\chi^2_{n-1;1-\alpha/2}} \right]
\end{equation}

é o IC bilateral para $\sigma^2$ com confiança de $1 - \alpha$.

\subsection*{5.3 Problema para duas amostras}

Considere a dedução de IC pela abordagem pivotal. Para uma função paramétrica desconhecida $K(\theta)$, assuma que temos um estimador $\hat{K}(\theta)$ que é função de uma estatística suficiente (mínima) para $\theta \in \Theta \subset \mathbb{R}^p$.
