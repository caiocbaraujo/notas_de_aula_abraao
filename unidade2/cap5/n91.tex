\textbf{Q(55)} Sejam $X_1, \ldots, X_n$ uma amostra de $X \sim U(0, \theta)$ para $\theta > 0$ desconhecido.  
Encontre o estimador intervalar bilateral para $\theta$ com confiança de $1-\alpha$.

\textbf{Solução}  
A estatística $T(X) = X_{(n)}$ é suficiente mínima para $\theta$ com densidade
\[
f_T(t; \theta) = \frac{n}{\theta^n} t^{n-1}
\]
